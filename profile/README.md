同人クラブ「中村音楽工業」のGitHubリポジトリ。ここではクラブの本業である音楽やアダルトゲームの制作を支える、周辺技術を公開するよ。

# ライセンス
フルスクラッチで開発したものは、基本的に三条項BSDライセンスを採用する。フォークしたものは、コピーレフトなら元のライセンスに、そうでないならケースバイケースとしている。

Kamuriki LGXは、固有の部分を三条項BSDライセンスとし、その他の部分はDebian自由ソフトウェアガイドライン（DFSG）に従う。

# 開発について
本クラブのOSSは全て、クラブの周辺技術として開発しています。 **そのため、コードは開かれている（＝オープンソースイニシアティブや自由ソフトウェア財団に認可されたライセンスの下で公開している）が、開かれた開発をしているわけではありません。** つまり、開発は我々のみが行なっています。

**外部からのプルリクエストは、原則お断りのコメントと共にクローズします（たまにコメントなしの時もある）ので、悪しからず。**

※時雨堂様やLuaの開発方針を参考にしています。
- [時雨堂様の該当する項目](https://github.com/shiguredo/oss)
- [LuaのFAQ](https://www.lua.org/faq.html#1.8)

なお、以下のリポジトリのみ、外部からのプルリクエストを受け付けています。
- [unifetch](https://github.com/nmimusic/unifetch)

A GitHub repositories of a dojin club, Nakamura Musical Industries. Here, we publish the supporting technologies what underpin our core business (production of music and hentai games).

# Licence
Software developed entirely from scratch is generally licensed under the 3-clause BSD licence. For projects that we forked, if the licence is copyleft, the original licence applies; otherwise, the decision is made on a case-by-case basis.

Kamuriki LGX is licensed under the 3-clause BSD licence for developed by us, whilst the remaining parts comply with the Debian Free Software Guidelines (DFSG).

# About Development
All of this club's OSS is developed as part of our supporting technology. **Consequently, the code is open (i.e. published under licences approved by the Open Source Initiative or the Free Software Foundation), but not openly developed.** In other words, development is carried out exclusively by our members.

**Please note that we generally close pull requests from external contributors with a comment stating that we do not accept them (though occasionally we may close them without comment). We apologise for any inconvenience this may cause.**

※We use development policies of Shiguredo and Lua as a reference.

Please note that we accept pull requests from external sources for the following repositories.
